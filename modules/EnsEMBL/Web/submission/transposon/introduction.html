Transposon Submission
  

Welcome to the Mosquito Transposon Submission Page

The transposon submission page allows submitted transposons to be mapped to the Anopheles genome and shown within the ensembl genome browser. The transposons will be loaded into ensembl every anopheles release and a dedicated view page will be designed to view detailed information about the transposon annotation. This submission tool is still under developmet, if you have any problems or suggestions ow to improve the submision or view of transposons please contact us

As the transposon sequences will be mapped by ensembl there is no need to submit location information, only an identifier, family and full transposon sequenece is required for each submission. Other anotation options will be added to the submission form at a later date. The transposon identifier should follow the guidelines and the family name should be selected from the drop down list. If the family name is not listed then select 'other'; after the transposon has been submited the family name can be added to the family list. Because the family name will be used for other submissions please be careful when entering the family name and use the naming guidelines.

For more information about this area please contact our helpdesk.
