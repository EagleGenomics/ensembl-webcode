Welcome to the Mosquito submission pages

The submission pages allows biologists to submit Anopheles genename assignments and transposable elements which they have studied to Ensembl. The submission tool allows the information to be mapped to the mosquito assembly and displayed within the ensembl browser. The annotations submitted via this tool will be updated into ensembl every release and during the update we will do everything possible to recapitulate the date submitted.

Login details created for the genename submission tool can also be used to submit transposons. The transposon submission tool is still under developmet, if you have any problems or suggestions to improve the submission of transposons please please contact our helpdesk.
