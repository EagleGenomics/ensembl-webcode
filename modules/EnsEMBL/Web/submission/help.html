User Registration
  • The fields in red are compulsory.
  • The email address has to be valid as you will receive your password via this address.
  • Click Register when finished.

User Logon
  • Enter you e-mail address and password (received via e-mail)
  • Click Logon.
  • You will see your logon user name appear at the top of the navigation bar

Submit a New Gene Symbol

To go to the submission page click on the New Gene link. Before submitting a genename please carefully read the Introduction and the Guidelines pages.

  • A gene submission page is valid for only one symbol. If you want to submit another gene symbol click on the New Gene link after submitting a gene.
  • The gene symbol and description fields are compulsory when submitting a genename but we strongly advise you to add as much information as possible.
  • Once you have added the required information, click the Add button, the submitted gene details will then be displayed. You may add further information such as additional cross references, synonyms and sequences by clicking on the Add [item] button, a new box will appear for that [item] field allowing you to add further entries.

Editing a Gene Submission

To update a gene that you have already submitted, click on the link My Genes. From this page select the gene that you would like to update.

  • A summary of the gene you wish to update will be shown, click on the Edit button in the bottom right corner of the gene box.
  • The information already submitted will appear on the page. You will then be able to modify the information or delete data already submitted.
  • To change the data, simply edit the information already submitted.
  • To remove data, delete your existing submission from the relevant text box.
  • To delete a whole submission, remove the gene symbol from the Proposed Symbol text box or click on the Delete button on the gene summary page and confirm the deleteion.

In all of the above cases you must click the Update button to store your changes for that gene.

Searching for a Gene

 To find a submitted gene click on Find Gene in the navigation bar.
  • Enter the search word(s) you would like to search by.
  • Select the type of search to search by
  • Click the Search button
  • If you would like to see all the currently submitted genes click the link at the top of the page

Submit a Transposon

To go to the submission page click on the New Transposon link in the navigation bar on the left of the page. Before submitting a transposon please carefully read the Introduction and the Guidelines pages for submitting advice.

  • A transposon submission form is valid for one transposon.
  • The transposon, family and sequence fields are compulsory when submitting a transposon but we strongly advise you to add as much information as possible.
  • When entering a family name for a transposon please select from the drop down list, if the family does not apear in this list please select 'Other'. When the transposon has been submitted you will be asked to enter a new family name. Please be careful when entering a family name and only enter an approved family (please use alpha-numeric characters only) as this will be used for future submissions
  • Once you have added the required information, click the Add button, the submitted transposon details will then be displayed.

Editing a Transposon

To update a Transposon that you have already submitted, click on the link My Genes from the navigation bar on the left of the screen. From this page select the transposon that you would like to update.

  • A summary of the transposon you wish to update will be shown, click on the Edit button in the bottom right corner of the screen.
  • The information already submitted will appear in an editable form. You will then be able to modify the information or delete any data already submitted.
  • To change the data, simply edit the information and click the update button.



Finally if you have any queries or ideas for improvement please contact us.
