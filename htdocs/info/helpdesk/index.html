Help

Online Help

When browsing Ensembl use the Help button at the top right hand side of the page for a pop-up context-sensitive help, keyword and history searches. You might also find the glossary helpful.

Contacting HelpDesk

For website bug reports, data problems and general questions about Ensembl, please check the documentation in HelpView. If you cannot find an answer, use the Contact Helpdesk button in HelpView or send an email to helpdesk@ensembl.org.

Courses and Tutorials

We have a range of online resources to help you learn about Ensembl, and we also run workshops - see the links in the left-hand column for more information.
