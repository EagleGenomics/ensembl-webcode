General Documentation and Information

  • The Ensembl project provides data sets resulting from an automated genome analysis and annotation process. In addition to Ensembl and EST gene sets, the project also provides other data sets on a genome-wide scale.

    More Documentation...

Data Downloads

  • All data sets generated by the Ensembl project are freely available to download from the ftp.ensembl.org site. Please see also the disclaimer.

  • Ensembl provides sequence databases of gene, transcript and protein predictions suitable for local installations of sequence similarity search systems, as well as database table dumps of all databases underlying the Ensembl system.

    More on FTP Downloads...

Comparative genomics and proteomics data

  • Ensembl produces comparative genomics data using BlastZ-net, translated blat, MLAGAN and gene homology analysis.

    More on comparative data...

Data Mining

  • Use BioMart our large-scale data mining tool for more advanced data queries. The tool goes way beyond the simple data sets we provide via the FTP site and you can export the data in HTML, Microsoft Excel and various text formats suitable for import into other database or analysis systems.

    More on BioMart...

Data Export

  • For each species Ensembl provides a dedicated ExportView page. This data display is especially designed for small-scale data exports in HTML, text or zipped format.

Data Import

  • View small custom data sets in Ensembl.

  • Set up a DAS server to display larger data sets in a more stable and flexible manner.
