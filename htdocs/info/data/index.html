Documentation and Information

  • The Ensembl project provides data sets resulting from an automated genome analysis and annotation process. In addition to Ensembl and EST gene sets, the project also provides other data sets on a genome-wide scale. More ...

Data Downloads

  • All data sets generated by the Ensembl project are freely available to download from the ftp.ensembl.org site. Ensembl provides sequence databases of gene, transcript and protein predictions suitable for local installations of sequence similarity search systems, as well as database table dumps of all databases underlying the Ensembl system. Data Mining

  • Use BioMart our large-scale data mining tool for more advanced data queries. The tool goes way beyond the simple data sets we provide via the FTP site and you can export the data in HTML, Microsoft Excel and various text formats suitable for import into other database or analysis systems. More ...

Export data

  • For each species Ensembl provides a dedicated ExportView page. This data display is especially designed for small-scale data exports in HTML, text or zipped format.

Searching

  • Text search to look for gene symbols, descriptions, marker names, ...
