Ensembl Functional Genomics

Database Description

The Ensembl Funcgen database and API are developed and maintained by the Ensembl Regulation team. The main focus of the 'funcgen' database is on experimental data which may elucidate transcriptional regulation. The schema was originally designed to support ChIP on Chip tiling array experiments but soon expanded to support other sequence based technologies. The original requirements supported use as a local data storage and analysis platform, as such the funcgen data bases supports multiple coordinate systems. The database content includes:

  • The Ensembl Regulatory Build - Regulatory features generated from a combined analysis of all of the above.
  • Experimental data and meta data from functional genomics experiments using the following technologies:
    • Chromatin immunoprecipitation using tiling arrays (ChIP-chip)
    • Chromatin immunoprecipitation sequencing
    • Other techniques identifying open chromatin e.g. DNase sequencing, FAIRE
  • Experimental data on epigenomic modifications and DNA binding proteins:
    • Histone modifications
    • Transcription factors e.g. CCCTC binding factor(CTCF) etc.
    • RNAPollII
  • Externally curated data with a regulatory focus:
    • cisRED - Regulatory motifs
    • miRanda - miRNA targets
    • VISTA - Enhancers
  • Array designs
    • For various array types e.g. Affy IVT/ST, Illumina Whole Genome/Infinium, Codelink, Agilent etc
    • Ensembl transcript annotations for Expression arrays.

Perl API

A comprehensive Perl Application Programme Interface (API) provides efficient access to the Ensembl Compara database.

  • EnsEMBL Perl API Installation: A step-by-step installation guide for all Ensembl Perl APIs.
  • Funcgen Perl API Documentation: A complete reference to the objects and methods used in the Funcgen database API.
  • Funcgen Perl API Tutorial: An introduction to the underlying concepts of the Funcgen database API.
