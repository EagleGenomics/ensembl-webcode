Generic File Format - Definition and supported options

The "generic" format supported by Ensembl is a very simple format similar to BED. It has three required fields and one optional, and supports no track options.

Required fields

  1. Seq region - name of the sequence region. Any valid seq_region_name can be used.
  2. Start - start position of the feature in Ensembl coordinates
  3. End - end position of the feature in Ensembl coordinates

Optional field

  1. ID - the name or other identifier for this feature (string).
