Ensembl User Accounts

Why User Accounts?

Please note that it is not necessary to register in order to gain access to Ensembl data. All our data remains free access, as it always has done.

Only external data uploaded by individuals or groups will be restricted to those users, in accordance with the policies described in our privacy statement.

Our user account system is designed to make this website more useful to visitors. We anticipate it will be particularly valuable in the following scenarios:

  • Users who move from one computer to another frequently, e.g. between lab and office, or between more widely-separated locations.
  • Users who share a computer, e.g. in a lab situation, but use different web features or different parts of the web site.
  • Groups of users who want to share settings, as an aid to collaboration
  • Groups of users who need to restrict access to confidential data but do not have the resources to set up their own copy of Ensembl.

Account features

Please note that this list may not be exhaustive, as the user account system is under continuous development.

  • Personal settings:
    • Save bookmarks
    • Save page configurations, e.g. which tracks are turned on in ContigView
    • Customise the homepage with your favourite species
    • Filter news headlines on the homepage by species and/or topic
  • Groups:
    • Join open groups (e.g. Cytogenetics) with pre-configured settings to help you get started
    • Request to join a restricted (moderated) group
    • Create your own restricted group
