Module EnsEMBL::Web::Configuration::Location

This script contains modules for configuring ``location'' based scripts. Currently configured in this script are:

contigview
cytoview
miscsetview
ldview
ldtableview
anchorview

Additionally there is the context_menu method for creating the ``Location'' based links.

Support functions

There is just one support function on the EnsEMBL::Web::Configuration::Location object:

top_start_end
